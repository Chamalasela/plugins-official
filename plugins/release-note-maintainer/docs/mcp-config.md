# MCP Configuration

> **Note:** The Release Note Maintainer Agent does **not** use the GitHub MCP server. **Git** is used for commit history, tag resolution, and repository structure on all hosts; **`gh`** and the Azure DevOps REST API are used only to list merged PRs and publish the release notes — see [`docs/platform-setup.md`](./platform-setup.md).

General platform setup (GitHub `gh` auth, Azure DevOps PAT) is documented in [`docs/platform-setup.md`](./platform-setup.md). Credential details are in [`docs/git-auth.md`](./git-auth.md).
