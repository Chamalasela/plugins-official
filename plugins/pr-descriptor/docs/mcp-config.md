# MCP Configuration

> **Note:** The PR Description Agent does **not** use the GitHub MCP server. **Git** is used for analysis on all hosts; **`gh`** and the Azure DevOps REST API are used only to resolve the PR and replace its title/description — see [`docs/platform-setup.md`](./platform-setup.md).

General platform setup (GitHub `gh` auth, Azure DevOps PAT) is documented in [`docs/platform-setup.md`](./platform-setup.md). Credential details are in [`docs/git-auth.md`](./git-auth.md).
